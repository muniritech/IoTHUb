# IoTHub

## Description
IoTHub is a web-based IoT management system built using ASP.NET Core 8.0 and React. It provides a platform for managing IoT devices, featuring JWT-based authentication and a MongoDB backend for data persistence.

## Execution Context
- **Backend**: ASP.NET Core Web API / MVC.
- **Frontend**: React application (located in `bin/Debug/net8.0/frontend/`).
- **Database**: MongoDB.
- **Authentication**: JWT Bearer tokens.

## Dependency Management
- **Backend**: Managed via `IoTHub.csproj` (NuGet).
- **Frontend**: Managed via `package.json` (NPM).
